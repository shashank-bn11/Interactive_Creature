# Interactive Embedded Animal Creature 

## Description
This project is an interactive embedded system built using Arduino Uno. 
It senses human presence, light conditions, and touch, and reacts using LEDs, 
a buzzer, a servo motor, and an OLED display.

## Features
- Ultrasonic sensor detects human presence
- LEDs and buzzer react when someone comes close
- LDR detects light/dark and updates OLED display
- Touch sensor controls servo motor movement
- OLED shows Awake / Sleeping states

## Components Used
- Arduino Uno (ATmega328P)
- HC-SR04 Ultrasonic Sensor
- LDR (Photoresistor)
- Touch Sensor
- Servo Motor (SG90)
- OLED Display (SSD1306, I2C)
- LEDs, Buzzer, Resistors

## Programming Language
- Embedded C (Arduino C)
